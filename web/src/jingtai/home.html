使用教程

快速上手指南

三分钟完成配置，立即开始使用 Claude

1

注册并登录账户

访问 http://1.95.142.151:3000，注册一个账户（请记住用户名，方便找回），注册完成后登录。

2

兑换额度

点击「钱包管理」→「兑换额度」，将人民币兑换为美元额度。兑换完成后可在钱包查看当前余额。

3

创建 API 令牌

点击「令牌管理」→「添加令牌」。名称随意填写，令牌分组可选择默认或 Claude Code 分组（1.5倍倍率，带缓存，更稳定）。其他选项默认即可，最后点击提交。

Claude Code 分组倍率更高但支持 1M 上下文和缓存，整体使用体验更顺滑。
4

复制使用 API Key

复制令牌时注意检查前后是否有空格，确保准确复制。URL 地址请看下方：

http://1.95.142.151:3000
注意：我们是 http 不是 https，别写错了！
